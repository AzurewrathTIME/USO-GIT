
# Proyecto: Multiplicación de Matrices y Medición de Tiempo

## 📌 Descripción
Este proyecto implementa la multiplicación de matrices 3x3 utilizando diferentes lenguajes de programación (C y Python).  
Además, se mide el tiempo de ejecución para comparar el rendimiento de los programas.

---

## 📁 Archivos del proyecto

- `mat.c` → Implementación en C de la multiplicación de matrices.
- `matrices.py` → Implementación en Python de la multiplicación de matrices.
- `time.py` → Versión en Python con medición de tiempo de ejecución.
- Otros archivos de prueba relacionados con matrices.

---

## ⚙️ Funcionamiento

El programa realiza lo siguiente:

1. Declara dos matrices 3x3.
2. Calcula la matriz resultante mediante multiplicación.
3. Imprime la matriz final.
4. Mide el tiempo de ejecución del proceso.

---

## ⏱️ Medición de tiempo

- En C se utiliza `clock()` de `<time.h>`
- En Python se utiliza `time.time()`

Esto permite comparar el rendimiento entre ambos lenguajes.
